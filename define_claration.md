## 类的结构

### Berth

| x,y            | 船舶左上角的坐标         |
| :------------- | ------------------------ |
| transport_time | 该泊位到虚拟点的时间     |
| loading_speed  | 一帧可以装几个物品到船上 |



### Robot

| x,y               | 当前位置                                 |
| :---------------- | ---------------------------------------- |
| target_x,target_y | 目标位置                                 |
| status            | 状态（0表示恢复状态（碰撞），1正常运行） |
| goods             | 是否携带物品（0没有，1有）               |

### Boat

| num    | 船上当前有多少货物                                          |
| ------ | ----------------------------------------------------------- |
| pos    | 目标泊位，如果目标是虚拟点则pos为-1                         |
| status | 0表示移动，1表示装货状态或运输完成状态，2表示泊位外等待状态 |
|        |                                                             |
